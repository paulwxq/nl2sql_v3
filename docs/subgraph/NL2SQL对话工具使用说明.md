# NL2SQL 对话工具使用说明

## 快速开始

### 启动对话（推荐）

```bash
python scripts/nl2sql_subgraph_cli.py
```

就这么简单！会进入持续对话界面：

```

================================================================================

欢迎使用！我可以帮您把自然语言问题转换为SQL语句。

💡 提示：
  - 直接输入问题，按回车键提交
  - 输入 'clear' 清屏
  - 输入 'exit' 或 'quit' 退出
  - 按 Ctrl+C 也可以退出

--------------------------------------------------------------------------------

👤 您: 
```

SELECT 
    ds.store_name,
    SUM(fs.amount) as total_amount
FROM public.fact_sales fs
JOIN public.dim_store ds ON fs.store_id = ds.id
WHERE 
    fs.order_date >= '2024-09-01' 
    AND fs.order_date < '2024-10-01'
    AND ds.store_name IN ('京东便利', '全家')
GROUP BY ds.store_name
ORDER BY total_amount DESC

## 单次查询

如果只需要查询一次：

```bash
python scripts/nl2sql_subgraph_cli.py "请对比一下9月份京东便利和全家这两个公司的销售金额"
```

## 输入格式（自动）

您只需输入自然语言问题，工具会自动包装成：

```python
{
    "query": "请对比一下9月份京东便利和全家这两个公司的销售金额",
    "query_id": "q_20251102_143052_a1b2c3d4",  # 自动生成
    "dependencies_results": {},
    "user_query": "请对比一下9月份京东便利和全家这两个公司的销售金额"
}
```

## 对话命令

在对话模式中：

- **提问**：直接输入问题，按回车
- **清屏**：输入 `clear`
- **退出**：输入 `exit`、`quit`、`q` 或按 `Ctrl+C`

## 日志查看

所有对话记录都保存在：

```
logs/sql_subgraph.log
```

实时查看日志（Windows）：
```powershell
Get-Content logs/sql_subgraph.log -Wait -Tail 50
```

## 特点

✅ **持续对话** - 像ChatGPT一样连续提问
✅ **自动包装** - 无需手动构造JSON格式
✅ **智能解析** - 理解时间、维度、指标
✅ **三层验证** - 确保SQL正确性
✅ **友好输出** - 清晰的对话式界面
✅ **完整日志** - 所有过程记录在日志

## 故障排查

### 数据库连接失败
检查 `.env` 文件配置和服务状态

### API密钥错误  
检查 `DASHSCOPE_API_KEY` 环境变量

### 模块导入错误
确保在项目根目录运行
